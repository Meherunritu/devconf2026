# DevConf 2026 Landing Page

A responsive landing page for a fictional developer conference built using only **HTML5** and **CSS3**. This project recreates a modern conference website with a clean UI, responsive layouts and reusable CSS components.

---

## Preview

The landing page includes:

- Hero section with background image and call-to-action
- navigation bar
- Featured speakers section
- Pricing plans
- AI placeholder section: Hackathon
- Responsive footer

---
## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS Variables
- Responsive Design

## Responsive Design

The landing page is responsive across desktop, tablet and mobile devices using:

- CSS Grid
- Flexbox
- Media Queries
- Relative sizing
- Fluid typography with `clamp()`
